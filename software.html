Software

The following software systems are currently available for the SamurAI 3x3 game.

  • SamurAI 3x3 Game Management System: Manages a SamurAI 3x3 game according to its rules, invoking AI programs to control samurai. Several sample AI programs are also included. The system and sample programs are written in C++11.
  • Game Log Visualizer: Graphical visualizer for game logs output by the game management system. This tool is written in Java.
  • More sample AI programs written in languages other than C++ are in preparation.
